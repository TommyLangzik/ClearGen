# 🖼️ ClearGen: v1.0.0 Stable

**ClearGen** is a premium, zero-friction utility designed for creators who need transparent PNG assets fast. Built with a focus on tactile feedback and high-end aesthetics, it turns the chore of asset creation into a "joy-hopping" experience.

![ClearGen UI Preview](https://via.placeholder.com/800x450.png?text=ClearGen+Premium+Interface+Preview) <!-- Replace with a screenshot of your app once hosted! -->

## ✨ Features
- **No-Install Portability**: A single [.html] file (ClearGen.html) that works in any modern browser.
- **Dynamic Themes**: 
  - 🟩 **Emerald Zen**: A lush, professional bright 'Light Mode'.
  - 🟥 **Migraine-Friendly Vader-Red**: A deep, high-contrast 'Dark Mode' designed for night-time comfort.
- **Tactile UI**: Smooth squash-and-stretch "jiggle" clicks and "joy-hop" rollover animations.
- **Resolution Library**: Quickly select from professional dimensions (Social Media, Web, Print) or swap custom Width/Height instantly.
- **Prompt Widgets**: Expandable cards with "click-to-copy" prompts for your creative workflow.
- **Glassmorphism Design**: High-end translucent surfaces and a modern conic-gradient checkerboard preview.

## 🚀 Getting Started
ClearGen requires **no installation** and has **zero dependencies**.
1. **Download** the 'ClearGen.html' file.
2. **Open** it in any modern web browser (Chrome, Edge, Firefox, Safari).
3. **Generate**: Set your dimensions and hit "Generate PNG".

*Pro-Tip: You can "Install" ClearGen as a PWA (Progressive Web App) by clicking the install icon in your browser's URL bar for a native desktop experience.*

## 🛠️ Built With
- **HTML5 Canvas**: Pure client-side PNG generation.
- **Vanilla CSS3**: Custom design tokens and layered animation system.
- **Tailwind CSS**: Modern utility-first layout.
- **Google Fonts**: Inter (900-weight for that premium feel).
